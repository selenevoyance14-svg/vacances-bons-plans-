Le cadenas TSA est indispensable pour les vols vers les Etats-Unis : la TSA (Transportation Security Administration) peut ouvrir votre valise sans casser le cadenas grace a un passe-partout. Mais il est aussi utile partout dans le monde pour securiser vos bagages.

## Pourquoi choisir un cadenas TSA

- **Obligatoire USA** : la TSA ouvre les bagages suspects — sans cadenas TSA, ils cassent le votre
- **Universel** : fonctionne comme un cadenas classique partout dans le monde
- **Securite** : dissuade le vol dans les aeroports, hotels et auberges de jeunesse
- **Types** : a combinaison (pas de cle a perdre) ou a cle (plus rapide a ouvrir)
