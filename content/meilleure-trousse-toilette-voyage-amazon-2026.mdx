La trousse de toilette de voyage est un accessoire qu'on sous-estime souvent. La bonne trousse garde vos affaires organisees, protege contre les fuites et se suspend dans les petites salles de bain d'hotel. Voici les meilleures sur Amazon.

## Comment choisir sa trousse de toilette de voyage

- **Suspendue** : se fixe a un crochet ou une porte, ideal dans les petites salles de bain
- **Etanche** : doublure impermeabilisee pour eviter les catastrophes en cas de fuite
- **Compartiments** : pochettes transparentes pour voir le contenu, filets pour flacons
- **Taille** : compacte pour un week-end, grande pour un long voyage
- **Norme avion** : pochette transparente 20x20 cm pour les liquides en cabine
