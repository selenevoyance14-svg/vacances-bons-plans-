Pour les trekkeurs, campeurs et voyageurs hors des sentiers battus, le chargeur solaire est la solution pour garder ses appareils charges sans prise electrique. Voici les meilleurs panneaux solaires portables sur Amazon.

## Comment choisir son chargeur solaire

- **Puissance** : 10-15W pour un telephone, 20-28W pour tablette + telephone simultanes
- **Type** : panneau seul (plus leger) ou panneau + batterie integree (plus pratique)
- **Pliable** : se replie pour le transport, se deplie au soleil
- **Ports** : USB-A et USB-C pour compatibilite maximale
- **Rendement** : les cellules monocristallines offrent le meilleur rendement (21-24%)
